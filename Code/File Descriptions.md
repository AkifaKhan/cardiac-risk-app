EDA and modelling of mortality on intraoperative variables.

Modelling of mortality via Firth Logistic Regression (to account for quasi-complete prediction problem). The implementation is unstable but works. Might need to tinker with hyper-parameters to ensure Hessian matrix is not problematic.



**Mortality\_Results\_on\_THI\_Testing\_Data.ipnyb** |

Mortality results on Tabba testing dataset using multiple models.
